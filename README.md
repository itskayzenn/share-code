## ğŸ—„ï¸ Setup Database (Supabase)

1. Buat project baru di [Supabase](https://supabase.com)
2. Jalankan SQL berikut di SQL Editor:

```sql
-- Membuat tabel snippets
create table snippets (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  title text not null,
  description text,
  code text not null,
  language text default 'text'
);

-- Matikan RLS untuk public access
alter table snippets disable row level security;
```

3. Ambil `Project URL` dan `anon/public` key dari Settings > API
4. Paste ke konfigurasi di file `index.html`

## ğŸ¤– Setup Telegram Bot

### Bot untuk Request Notifications
1. Buat bot baru dengan [@BotFather](https://t.me/botfather)
2. Copy token bot
3. Paste ke `TELEGRAM_BOT_TOKEN_REQUEST`

### Bot untuk Login Notifications
1. Buat bot lain dengan [@BotFather](https://t.me/botfather)
2. Copy token bot
3. Paste ke `TELEGRAM_BOT_TOKEN_LOGIN`

### Mendapatkan Chat ID
1. Kirim pesan ke bot Anda
2. Buka: `https://api.telegram.org/bot<TOKEN>/getUpdates`
3. Cari `"chat":{"id":123456789}`
4. Copy angka tersebut dan paste ke `TELEGRAM_CHAT_ID`

## ğŸ”’ Security Notes

## ğŸ› ï¸ Troubleshooting

### Snippets tidak muncul
- Cek koneksi Supabase
- Pastikan table `snippets` sudah dibuat
- Cek console browser untuk error

### Telegram notification tidak terkirim
- Pastikan bot token benar
- Pastikan chat ID benar
- Pastikan sudah kirim pesan ke bot minimal 1x

### Upload file tidak berfungsi
- Pastikan file size < browser limit
- Cek file extension yang didukung
- Cek console untuk error

### Raw view tidak berfungsi
- Cek URL format: `/#/raw/[snippet-title]`
- Pastikan snippet title exact match
