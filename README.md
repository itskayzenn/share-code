**Kayzen ShareCode** adalah platform berbagi snippet kode modern yang dirancang untuk kecepatan dan kemudahan akses. Dibangun dengan estetika *Cyber-Dark* dan integrasi penuh ke ekosistem Supabase.

## 🛠️ Tech Stack
- **Frontend:** Vanilla JS, Tailwind CSS, PrismJS (Syntax Highlighting).
- **Backend:** Supabase (Database & Auth), Vercel Serverless.
- **Integrasi:** Telegram Bot API untuk notifikasi Admin & Request.

## ✨ Fitur Utama
- 📱 **Mobile Optimized:** Dikembangkan khusus untuk kenyamanan pengguna mobile (Mobile Warrior).
- 📦 **Snippets CRUD:** Manajemen kode lengkap (Create, Read, Update, Delete).
- 🔔 **Live Announcement:** Notifikasi pengumuman langsung melalui ikon lonceng di navbar.
- 📤 **Smart Upload:** Mendukung drag & drop hingga 10 file sekaligus.
- 🔐 **Admin Dashboard:** Panel kontrol terproteksi dengan notifikasi login via Telegram.
- 📄 **Raw View:** Akses kode sumber dalam format plain-text secara instan.

## 🚀 Instalasi Cepat
1. **Clone & Setup:**
   ```bash
   git clone [https://github.com/itskayzenn/share-code-main.git](https://github.com/itskayzenn/share-code-main.git)
2. **Database:** Jalankan schema.sql di dashboard Supabase Anda.
3. **Environment:** Masukkan URL dan KEY Supabase Anda ke dalam konstanta CONFIG di index.html.
4. **Deploy:** Hubungkan repositori ke Vercel dan klik Deploy.
