Platform manajemen snippet kode modern dengan antarmuka futuristik, sistem notifikasi real-time, dan integrasi admin yang aman.

## 🛠️ Tech Stack
- **Frontend:** Vanilla JavaScript, Tailwind CSS 3.0
- **Backend:** Supabase (PostgreSQL & Auth)
- **Icons:** Font Awesome 6.4
- **Fonts:** JetBrains Mono & Poppins
- **Notifications:** Telegram Bot API Integration

## 🚀 Fitur Unggulan
- **Global Announcement:** Sistem pemberitahuan dinamis dari database.
- **Snippet Management:** CRUD lengkap untuk menyimpan script penting.
- **Admin Dashboard:** Panel kontrol khusus dengan notifikasi login ke Telegram.
- **Smart Search:** Pencarian snippet instan tanpa reload.
- **Responsive UI:** Dioptimalkan untuk perangkat mobile dan desktop.

## 🔧 Instalasi & Deploy
1. **Database:** Buat tabel menggunakan skema SQL yang disediakan.
2. **Environment:** Masukkan `URL` dan `KEY` Supabase ke dalam file `index.html`.
3. **Deploy:** Hubungkan repositori ke Vercel dan aktifkan fitur Rewrites.
